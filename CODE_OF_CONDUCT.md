# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal attacks are not
tolerated, in issues, pull requests, or any project space.

**Reporting:** raise concerns with the maintainer, Jurjen Stellingwerff
(<jstellingwerff@betterbe.com>). Reports are handled confidentially.

The full text and enforcement guidelines are at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
